# Footsies Dojo

差し合い（footsies）特化の2D格闘ゲーム。格闘ゲームの駆け引きの中から「差し合い」だけを抽出し、間合い管理・技振り・差し返しの読み合いに集中できるシンプルな対戦ゲームです。

**プレイはこちら:** https://reisun.github.io/footsies-dojo/

## 操作方法

| キー | アクション |
|------|-----------|
| `A` | 左移動 |
| `D` | 右移動 |
| `S` | しゃがみ |
| `S` + 後ろ方向 | しゃがみガード |
| 後ろ方向入力 | 立ちガード（後ろ歩き中） |
| 前方向 2回素早く入力 | ダッシュ（12F受付猶予） |
| `J` | 小攻撃 |
| `K` | 中攻撃 |
| `L` | 大攻撃 |
| `H` | ヒットボックス表示切替 |

## ゲームシステム

### 基本ルール
- 横軸のみの1ライン差し合い
- HP制、ラウンド制（2本先取）
- ラウンド制限時間: 60秒
- CPU対戦（将来の対人戦を見越した設計）

### 技のフレームデータ

| 技 | 発生 | 持続 | 硬直 | ダメージ | 削り | リーチ |
|----|------|------|------|----------|------|--------|
| 小攻撃 | 4F | 3F | 8F | 60 | 5 | 55px |
| 中攻撃 | 8F | 4F | 14F | 100 | 10 | 80px |
| 大攻撃 | 20F | 5F | 22F | 160 | 15 | 110px |

### 技の特徴
- **小攻撃**: 発生が早く隙が少ない。牽制・歩き止め向き
- **中攻撃**: バランス型。置き技・差し返しの主力
- **大攻撃**: 振りかぶりの予備動作あり。高リスク高リターン。ヒット時はダウンを奪える

### ガードシステム
- **立ちガード**: 後ろ方向に歩くことでガード成立
- **しゃがみガード**: `S` + 後ろ方向でガード成立（`S`のみはしゃがみでガード判定なし）
- しゃがみ中は食らい判定が70%に縮小
- ガード時は削りダメージを受ける
- ガードされた攻撃側が後ずさりするため、ガード相手への連打ハメを防止

### ダッシュ
- 前方向キーを素早く2回入力（12フレーム以内）
- 速度: 9px/F、持続: 12F
- 無敵なし、ダッシュ中は攻撃不可

### 差し返し（カウンターパニッシュ）
- **条件**: 相手の大攻撃のリカバリー中に中攻撃をヒットさせる
- **効果**: ダメージ1.2倍 + ダウン発生
- 大攻撃の大きな予備動作と後隙を見て差し返しを狙う読み合いが核

### ダウン・起き上がりシステム
- ダウン時間: 30F、起き上がり: 15F（合計45F）
- ダウン中・起き上がり中は完全無敵（当たり判定なし）
- 起き上がりに大攻撃を重ねる起き攻めの読み合いが発生

## 駆け引きの構造

```
待ち（しゃがみガード）
  └─ 大攻撃の空振りを差し返せる
  └─ 負ける: 歩き/ダッシュで間合いを詰められる

前歩き / ダッシュ
  └─ 待ちに対して間合いを詰める
  └─ 負ける: 置き技（小技・中技）で止められる

小技 / 中技（置き技）
  └─ 歩いてくる相手を止める
  └─ 負ける: リーチ外から大技で潰される

大技
  └─ 置き技のリーチ外から叩く
  └─ 負ける: 予備動作が見えて差し返される
```

## CPU AI

### 難易度

| 難易度 | 反応速度 | 攻撃性 | 最適行動率 |
|--------|----------|--------|-----------|
| Easy | 20F | 30% | 30% |
| Normal | 12F | 50% | 55% |
| Hard | 5F | 65% | 80% |

### AI行動パターン
- **遠距離**: 積極的に前歩き・ダッシュで接近
- **中距離**: 攻撃・しゃがみガード・位置調整を混ぜた差し合い
- **近距離**: ガード重視、反撃、またはバックステップ
- **差し返し待ち**: しゃがみガードで待機し、相手の大攻撃を見てから中攻撃で差し返し
- **空振り狩り**: 相手の攻撃リカバリーを検知して即反撃

## 技術スタック

- **言語**: TypeScript

- **描画**: HTML5 Canvas（960x540px、60FPS）
