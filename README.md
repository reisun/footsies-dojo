# footsies-dojo

差し合い（footsies）特化の2D格闘ゲームを新規作成する。

## 仕様
- Vite + TypeScript + HTML Canvas でブラウザゲーム
- GitHub Pages デプロイ前提（gh-pages対応ビルド設定）
- キーボード操作：左手(A/D移動, W前ステップ) 右手(J小技, K中技, L大技)
- ドット絵風キャラクター（Canvas描画でシンプルな格闘家、ヒットボックスと一致するデザイン）
- CPU対戦のみ（将来の対人戦を見越し、入力処理とゲームロジックを分離設計）

## ゲームメカニクス
- 横軸のみの1ライン差し合い
- 各技にリーチ・発生フレーム・硬直フレーム・ダメージを設定
- 小技：速い/短い/隙少/低ダメージ — 牽制・歩き止め用
- 中技：中速/中リーチ/中隙 — 置き技・差し返し用
- 大技：遅い/長い/隙大/高ダメージ — 置き技潰し・リスクリターン用
- 前ステップ：素早く間合いを詰める（無敵なし、硬直あり）
- 後ろ歩きガード：技を出していない＆後ろ入力でガード成立（削りダメージ小）
- HP制、ラウンド制（2本先取）
- ヒットボックス/やられ判定の可視化トグル

## CPU AI
- 距離に応じた行動選択（遠距離：歩き/様子見、中距離：置き技/大技、近距離：小技連打）
- ランダム要素を含み、パターン化しにくい設計
- 難易度設定（反応速度・行動選択の最適度を調整）

## 画面構成
- タイトル画面（難易度選択）
- バトル画面（HPバー、タイマー、ラウンド表示）
- リザルト画面（勝敗表示、リトライ）
